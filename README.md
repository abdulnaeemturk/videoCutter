# videoCutter
This video cutting tool is free and open-source software. Anyone can use it for educational purposes, including students, teachers, and learners. It helps easily trim and edit videos for school projects, tutorials, and learning activities. Users are free to use and share it to support education.


# FFmpeg Web Video Editing Examples

This repository contains several **web-based video editing examples built with FFmpeg**.

The goal of the project is to demonstrate how FFmpeg can be integrated into a **PHP + JavaScript web application** to perform video processing tasks such as:

- Video cutting
- Segment selection
- Timeline editing
- Thumbnail generation
- Text overlays
- Icon overlays
- Video concatenation
